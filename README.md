# MC-RO
本作品 MC逆滲透 為本團隊於2020移動應用創新賽中投稿的 iOS APP 作品  

榮獲：
1. 2020移動應用創新賽 台灣區 二等獎
2. 中國高校計算機大賽 2020移動應用創新賽 大中華區 三等獎

主要提供功能是(需搭配後端伺服器操作)：
1. 將音樂音軌分離為：人聲、吉他、Bass、鋼琴、其他(提琴)
2. 匯入網址，從Youtube提取音樂，並分離音軌(僅供測試與練習，檔案請於24小時內刪除)
3. 即時樂隊自我練習錄音，分離音軌
